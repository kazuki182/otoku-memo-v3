# おとくメモ flat v15

スマホで使いやすい買い物価格管理アプリです。

## v15の追加機能

- 承認制ログインを追加しました。
- 初期管理者は `kazuki`、共通パスワードは `12345` です。
- 管理者 kazuki でログインすると、設定画面から家族用アカウント名を承認・解除できます。
- ページ上部へ戻る「↑ 上へ」ボタンを追加しました。
- 価格登録で「店舗名」と「支店名（〜店）」を分けて入力できます。
- 店舗名 + 支店名は、価格履歴では「ウエルシア 川口店」のように保存されます。

## 更新方法

GitHubに以下5ファイルを上書きしてください。

- index.html
- styles.css
- app.js
- schema.sql
- README.md

## Supabase更新

v15では `approved_accounts` テーブルを追加しています。
SupabaseのSQL Editorで `schema.sql` を再実行してください。

既存データは削除せず、必要なテーブル・列だけ追加します。

## 注意

この承認制ログインは簡易版です。静的Webアプリ上の共通パスワード方式なので、本格的な個人情報管理や決済には使わないでください。

## v16更新内容
- 課金制をやめ、無料アプリとして使う前提に変更
- 任意の「応援・投げ銭」ページを追加
- PayPay API連携ではなく、PayPay ID/送金先メモの表示・コピー・任意リンク表示に対応
- 管理者 kazuki のみ応援ページ内容を編集可能
- Supabase利用時は `support_settings` テーブルに応援ページ設定を保存

### 注意
PayPay公式の決済API連携ではありません。無料範囲で使うための簡易表示方式です。
