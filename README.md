# おとくメモ flat版 v7

GitHubのブラウザアップロードでフォルダ構造が崩れても使える、静的HTML版です。

## v7の変更

- HOME上部に「容量かんたん比較」を常時表示
- 買い物中に、容量と価格だけでA/B比較できるように変更
- 比較単位を ml / g / 個 / 枚 / ロール / 本 から選択可能
- 買い物リストはサブ機能のまま維持
- 商品・価格登録済みの商品から買い物リストへ追加

## ファイル

- index.html
- styles.css
- app.js
- schema.sql
- README.md

## 使い方

GitHubに5ファイルを上書きアップロードしてください。Vercelが自動再デプロイします。
Supabaseを使う場合は、schema.sqlをSQL Editorで実行してください。


## v8 変更点

- 商品カードから直接「価格を追加」できるようにしました。
- 価格登録時、前回使った店舗を自動入力します。
- 商品ごとによく使う店舗も記憶します。
- 店舗名入力に予測候補を追加しました。
- 埼玉・東京・千葉でも使いやすいスーパー、ドラッグストア、量販店チェーン候補を初期搭載しました。
- 価格だけ入力して登録しやすい簡単登録モードに寄せました。
