# おとくメモ flat版 v23

無料で使える買い物価格管理アプリです。

## v23の追加内容

- HOMEに「使い方ガイド」ボタンを追加
- ボタンを押すと、アプリ内に使い方説明を表示
- 商品登録、価格登録、AI貼り付け、買い時判定、家族共有の流れを説明
- 使い方ページから「商品登録」「価格登録」「買い物リスト」へ移動できるショートカットを追加
