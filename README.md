# おとくメモ flat v20

## 更新内容

- 最寄り駅を設定できるようにしました。
- 最寄り駅をもとに「〇〇店」「〇〇駅前店」などの支店候補を表示します。
- 店舗名は候補・プルダウン中心で選べるようにし、表記ゆれを減らしました。
- ダイレックス、ジョイフード、ミートミート、ジャパンミートなどの店舗候補を追加しました。
- 設定にChatGPTリンクを登録できるようにしました。
- HOMEから「ChatGPTを開く」ボタンで登録したリンクを開けます。
- 製品写真を登録できるようにしました。
- 商品カードに登録した製品写真を表示します。
